# Note Taking App

This repository contains a full-stack note-taking application built with Node.js, Express, and MongoDB.

## GitHub

Linked remote: https://github.com/rjametge-collab

## Setup

1. Install dependencies: `npm install`
2. Add a `.env` file with your MongoDB URI and JWT secret.
3. Run in development mode: `npm run dev`
