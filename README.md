# Note Taking App

Small Notes app --

This app is a modern, full-stack note-taking web application built with a dark purple aesthetic. It combines a clean, modern writing experience with a persistent MongoDB backend, making your notes available across sessions.

What you can do?

* Create, edit, and delete notes with a rich writing toolbar
* Organize notes by tags (Work, Idea, Personal, Design)
* Live search and tag filtering across all notes
* Auto-save with manual save support (Ctrl+S)
* Writing stats — word count, character count, line count
* Writing activity heatmap and streak tracking
* Export any note as a .md Markdown file
* Persistent storage via MongoDB + Mongoose
* REST API backend built with Node.js and Express
